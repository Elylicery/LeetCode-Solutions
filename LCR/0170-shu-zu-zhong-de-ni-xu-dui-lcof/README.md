# LCR 170. 交易逆序对的总数

## 思路：利用归并排序求逆序对

在归并排序的 **“合并”阶段**，当我们将两个**已排序的子数组**合并时，**可以一次性计算跨越左右两部分的逆序对数量。**

✨ 核心思想：
* 左半部分 [l...mid] 和右半部分 [mid+1...r] 都已排序。
*  当 left[i] > right[j] 时，左半部分从 i 到末尾的所有元素都 > right[j]（因为左半部分有序）。
所以，逆序对数量 += (mid - i + 1)

时间复杂度：O(n log n)（与归并排序相同）
空间复杂度：O(n)（辅助数组）
是否修改原数组：是（可拷贝避免）
